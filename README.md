# isakov-bench

Side-by-side visual benchmark comparing AI coding agents on creative HTML/CSS/JS prompts.

**Live:** [bench.isakov.io](https://bench.isakov.io)

## Thumbnails

Thumbnails are 10-second WebM videos instead of live iframes to avoid loading dozens of heavy pages at once. Recorded with Playwright at 2x resolution, then re-encoded with ffmpeg at 120 fps.
